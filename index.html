HTML 5 & CSS Exercise

Ok, so this one really allowed me to get my teeth into developing my CSS skills. I could brush up on all the skills I had already learnt and also expand my styling knowledge. Now I was learning how to override the default display values of my HTML with the CSS display property. How quickly now I could line up any elements set next to each other using the vertical align property.
